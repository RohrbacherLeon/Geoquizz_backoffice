# Geoquizz_backoffice
Backoffice de gestion des photos et des series pour l'application Geoquizz.

## Installation
Dans un premier temps, placez vous Ã  la racine du projet et lancez la commande suivante :
```
npm install
```

## Executer le projet
Lancez la commande suivante :  
```
npm run serve
```
Puis rendez vous sur l'adresse [http://localhost:8080](http://localhost:8080)
